# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-29

### Added
- Initial release of TRIBE v2 Persona Predictor
- Python backend with persona encoding system
- 6 pre-built personality personas (analytical, creative, emotional, social, pragmatic, tech_savvy)
- 9 reaction type predictions (Attention, Engagement, Valence, Arousal, Memory, Aesthetics, Novelty, Social, Curiosity)
- Brain-persona fusion module with cross-attention
- React frontend with neural interface design
- Interactive brain visualization component
- Drag & drop video upload
- Reaction dashboard with confidence meters
- Radar chart visualization
- Content optimization recommendations
- Comprehensive demo scripts and notebooks

### Features
- Demo mode (works without TRIBE v2 installed)
- Full TRIBE v2 integration (requires HuggingFace auth)
- Persona customization via Big5 traits or MBTI
- Reaction comparison across personas
- Human-readable insights generation

### Design
- Bioluminescent neural interface aesthetic
- Dark atmospheric theme with cyan/blue gradients
- Glass-morphism UI components
- Animated neural network visualizations
- Floating particle effects
- Scroll-triggered animations
